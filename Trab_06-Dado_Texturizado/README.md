## Trabalho 06 - Dado Texturizado

### Como rodar

1. Rodar o arquivo [Trab_06-Dado_Texturizada.py](Trab_06-Dado_Texturizada.py) (Python3).

**Left-Click** - Girar figura;
**Right-Clik** - Mover a figura;
**Mouse-Wheel** - Zoom;
